Description

This web application was created in order to be able to manage sports competitions, in which they can register teams, players, competitions and matches. In addition, you can view the table of positions of each competition and the results of the matches.

The application was developed with the Django framework, which is a framework that is based on the model-view-template architecture, which allows the separation of the logic of the application from the presentation of the data. In addition, it has a database in which the data of the teams, players, competitions and matches are stored.
